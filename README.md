# Trader Behavior Analysis

## Objective
Analyze trader behavior under different market sentiment regimes and propose actionable strategies.

## Files Included
- Trader_Behavior_Analysis.ipynb
- historical_data.csv
- fear_greed_index.csv

## Dataset
-Due to GitHub file size limitations (100MB max per file), the `historical_data.csv` file (46MB) is hosted externally.
-You can download it from the link below:
-(https://drive.google.com/drive/folders/15UhHg9pW_c1UuoqnX7jXDNiT5PFbmLR2?usp=sharing)
-After downloading, place the file in the root directory of the project before running the notebook.

## How to Run
- Install required libraries:
   pip install pandas numpy matplotlib scikit-learn

- Open Jupyter Notebook
- Run all cells

## Summary
- Sentiment impacts trade frequency and profitability
- High activity traders show higher risk exposure
- Two actionable risk management strategies proposed
- Predictive model : Logistic Regression
